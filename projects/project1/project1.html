프로젝트명 : TravelKkaebi
상세보기 및 GitHub

진행기간 : 22.08.18 - 22.09.16

진행 인원 : 5명

소개 : 다른사람과 여행가고 싶은 사람들을 위한 여행멤버 구인 & 여행 정보 커뮤니티 사이트


사용기술 : Java(11), React(18), SpringBoot(2.7.3), SpringSecurity, MyBatis, JPA, MySql, Amazon S3, Github


주요기능 : 회원가입/로그인, 게시판(댓글, 검색, 키워드, 이미지), 다중 이미지, 좋아요, 매너온도 등


역할 : 프로젝트 리더, DB설계, 메인화면 최신 게시물리스트, 같이가요 게시판 CRUD 외 검색기능 등(Back-End), 좋아요 기능, 매너온도 기능, 다중이미지, 내 여행게시판


배운점 :

- 처음 해보는 협업 프로젝트이지만 프로젝트 리더를 맡아 프로젝트를 진행하면서 프로젝트 전반을 기획, 설계, 개발하는 과정을 경험해봤습니다.

- 1차, 2차, 3차 정규화를 통해 DB를 설계하는 방법을 알게되었습니다.

- @Controller, @Service, @Repository, @Mapper 의 역할에 대해 명확히 알고 각 Bean마다 어떤 종류의 로직이 들어가야 하는지 알게 되었습니다.

- @RestControllerAdvice를 이용해 Spring의 모든 예외들을 한곳에 모아서 처리하는 방법에 대해 알게되었습니다.

- Optional클래스에 대해 알게되었고 .OrElseThrow등을 통해 Null값 관련 예외를 처리하는 법을 알게되었습니다.

- SpringSecurity와 JWT토큰을 이용해 사용자 검증이 필요한 기능의 검증로직을 구현하는 방법을 알게되었습니다.

- React의 Axios를 이용해 SpringBoot와의 통신을 구현하면서 각 데이터들을 어떤형태로 어떤방식으로 주고받는지에 대해 알게되었습니다.

- RestApi의 4가지 통신방식(Get, Post, Put, Delete)과 각 통신방식간의 차이점에 대해 알게되었습니다.

- 여러명의 사용자가 기능을 이용할 때, 어떠한 예외들이 발생하고 그것들을 어떻게 처리해야 하는지에 대해 알게되었습니다.
